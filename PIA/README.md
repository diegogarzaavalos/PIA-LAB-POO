PIA — Proyecto Integrador Académico

Esta carpeta contiene el Proyecto Integrador del curso (PIA). Aquí se debe colocar el código final, documentación, diagramas y cualquier recurso necesario para compilar y ejecutar el proyecto.

Estructura sugerida:
- src/       -> código fuente
- docs/      -> documentación y diagramas
- build/     -> binarios o artefactos de compilación (no subir binarios grandes)
- README.md  -> instrucciones específicas del PIA

Instrucciones rápidas:
- Lee este README para ver cómo compilar y ejecutar el proyecto.
- Usa ramas separadas para características importantes y abre PRs para revisión.
