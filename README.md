# PIA-LAB-POO

Repositorio con las prácticas y el proyecto integrador (PIA) del curso de Programación Orientada a Objetos (POO).

Estructura del repositorio:
- /practicas/Practica1 ... Practica11: carpetas con los ejercicios y programas de cada práctica.
- /PIA/: carpeta para el proyecto integrador (PIA).

Cómo contribuir:
1. Clona el repositorio:
   git clone git@github.com:diegogarzaavalos/PIA-LAB-POO.git
2. Crea una rama para tu cambio:
   git checkout -b feature/mi-practica
3. Agrega tus archivos dentro de la carpeta correspondiente (por ejemplo practicas/Practica3/)
4. Haz commit y push
   git add .
   git commit -m "Agregar archivos de PracticaX"
   git push origin feature/mi-practica
5. Abre un Pull Request para revisión.

Notas:
- Si subes archivos binarios o grandes (>100MB) usa Git LFS.
- Mantén los nombres de carpeta en el formato PracticaN (ej. Practica1, Practica2, ...).
